# SFX Import Target

Current runtime usage:

- Jump and pause cues come from `res://audio/sfx/Movement/` and `res://audio/sfx/General Sounds/`
- Coin, power-up, and menu cues come from `res://audio/sfx/General Sounds/`
- Some gameplay cues use Kenney sounds from `res://assets/art/kenney_new_platformer_pack/Sounds/`

If you replace the current choices, update the lookup table in `AudioDirector.cs`.
