Place imported art here.

- `res://assets/art/kenney_new_platformer_pack/`
- `res://assets/ui/kenney_ui-pack-pixel-adventure/`

Guidelines:

- Keep source packs grouped by publisher or pack name.
- Preserve an `atlas`, `sprites`, or `tiles` split only after import, not before.
