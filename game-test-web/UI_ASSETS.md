# UI Assets

This web build uses free third-party UI assets placed under `public/visuals/ui`.

## HUD And Menu Chrome

- Source: [UI Pack - Pixel Adventure](https://opengameart.org/content/ui-pack-pixel-adventure)
- Author: `Kenney`
- License: `CC0`
- Used for:
  - HUD bar shell
  - HUD cell shells
  - title and modal ribbons
  - primary and secondary button chrome

## Derived UI Files

The files under `public/visuals/ui` are small composites generated from the pack's thin-outline large tilesheet:

- `button-primary.png`
- `button-secondary.png`
- `ribbon-wide.png`
- `ribbon-compact.png`
- `hud-bar.png`
- `hud-cell.png`

## Raw Downloads

Original downloads are stored under `assets/ui/downloads/` for traceability.
