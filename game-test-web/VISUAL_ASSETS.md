# Visual Assets

This web build uses free third-party visual assets placed under `public/visuals`.

## Character Animation

- Source: [Kenney Platformer Characters](https://kenney.nl/assets/platformer-characters)
- License: `CC0`
- Used for:
  - player forms and pose-based animation
  - zombie ground-enemy animation

## Terrain And Structures

- Source: [Kenney Pixel Platformer](https://kenney.nl/assets/pixel-platformer)
- License: `CC0`
- Used for:
  - ground top and fill tiles
  - moving/falling platform visuals
  - compact non-humanoid enemy sprites

## Backgrounds

- Source: [Kenney Background Elements](https://kenney.nl/assets/background-elements)
- License: `CC0`
- Used for:
  - sky
  - cloud band
  - mountains
  - hills
  - tree silhouettes

## Raw Downloads

Original downloads are stored under `assets/visuals/downloads/` for traceability.
